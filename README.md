# Habit Tracker

A beautiful, GitHub-inspired habit tracker web app with a dark theme and contribution graph visualization.

![Habit Tracker Preview](https://via.placeholder.com/800x400?text=Habit+Tracker+Preview)

## Features

- **GitHub-inspired dark theme** — Clean, modern UI matching GitHub's aesthetic
- **Contribution graph** — 365-day heatmap showing your habit consistency
- **Streak tracking** — Track your current and longest streaks
- **Persistent storage** — All data saved to localStorage
- **No dependencies** — Single HTML file, runs anywhere

## Usage

1. Open `index.html` in any modern browser
2. Add habits using the input field
3. Click "Done" to mark today's habit complete
4. Watch your contribution graph grow!

## Tech

- Pure HTML/CSS/JS
- localStorage for persistence
- CSS Grid for the contribution graph
